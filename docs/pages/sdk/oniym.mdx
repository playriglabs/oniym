# Oniym class

The `Oniym` class is the main entry point for the SDK.

## Constructor

```ts
import { Oniym } from "@oniym/sdk";

const oniym = new Oniym(config?: OniymConfig);
```

### OniymConfig

```ts
interface OniymConfig {
  indexerUrl?: string; // default: "http://localhost:42069"
  rpcUrl?: string;     // default: public Base Sepolia RPC
  chainId?: 84532 | 8453; // default: 84532 (Base Sepolia)
}
```

## Read methods

| Method | Description |
|--------|-------------|
| [`resolve(name)`](/sdk/resolution#resolve) | All records for a name |
| [`getName(address)`](/sdk/resolution#getname) | Reverse lookup |
| [`getAddress(name, chain)`](/sdk/resolution#getaddress) | Single chain address |
| [`getAddresses(name)`](/sdk/resolution#getaddresses) | All chain addresses |
| [`available(name, tld)`](/sdk/resolution#available) | Availability check (RPC) |
| [`rentPrice(name, tld, duration)`](/sdk/resolution#rentprice) | Price in wei (RPC) |
| [`getTLDs()`](/sdk/utilities#gettlds) | List all supported TLDs |

## Write methods

| Method | Description |
|--------|-------------|
| [`register(options, walletClient)`](/sdk/registration) | Full commit-reveal registration |
| [`setAddress(name, chain, address, walletClient)`](/sdk/registration#setaddress) | Set a chain address |
| [`setText(name, key, value, walletClient)`](/sdk/registration#settext) | Set a text record |
